# Geosoft Project

##Server Starten
- Im hauptverzeichnis des  Projekts  datenbank  Starten
`mongod --dbpath data`
- Im Hauptverzeichnis  Node Server  Starten `npm  start`
  - Im  Bedarfsfall  mit npm vorher  dependencys  installieren `npm install`
